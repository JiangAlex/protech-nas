## 登入

- 網址：`http://<NAS_IP>`（預設 port 80）
- 預設帳號：`admin`
- 預設密碼：`admin123`

> ⚠️ 首次登入後請立即修改密碼。

## 儀表板

- 顯示 CPU、記憶體、磁碟使用率（圓餅圖）
- 系統資訊：主機名稱、OS、運行時間、網路流量
- 歷史趨勢圖表（24 小時 CPU/記憶體折線圖）
- 右上角可切換自動刷新開/關

## 檔案管理

### 基本操作
- **瀏覽**：點擊資料夾名稱進入，麵包屑導航可快速跳轉
- **上傳**：點「上傳」按鈕或拖拽檔案
- **下載**：選取檔案 → 點「下載」
- **新資料夾**：點「新資料夾」→ 輸入名稱 → 建立

### 進階操作
- **搬移**：選取檔案 → 點「移動」→ 輸入目的路徑
- **複製/貼上**：選取 → 點「複製」→ 切換到目的目錄 → 點「貼上」
- **重新命名**：雙擊檔案名稱 → 直接修改 → Enter 確認
- **壓縮**：選取多個檔案 → 點「壓縮」→ 選格式（ZIP/TAR.GZ）
- **解壓**：選取壓縮檔 → 點「解壓」→ 指定目的路徑
- **預覽**：單擊檔案名稱，右側 drawer 顯示預覽
  - 圖片：直接顯示
  - 文字/程式碼：顯示內容（限 100KB）
  - PDF：嵌入檢視
  - 影片：播放器
- **分享連結**：點「分享」→ 設定有效時間 → 可選密碼 → 產生連結

## 儲存管理

### 磁碟 Tab
- 顯示所有磁碟和分割區資訊
- **格式化**：點「格式化磁碟」→ 選擇裝置 → 選檔案系統（ext4/XFS/Btrfs/exFAT）→ 輸入裝置名稱確認
  - ⚠️ 系統碟（掛載在 `/` 的磁碟）不會出現在列表中
  - ⚠️ 格式化不可逆，所有資料將被清除

### S.M.A.R.T. Tab
- 選擇磁碟 → 顯示健康狀態、溫度、通電時數、屬性表
- 可執行短測試

### 分割區 Tab
- 顯示非系統碟的分割區
- **建立分割區**：選磁碟 → 設定大小 → 類型（Primary/Logical）
- **刪除分割區**：點「刪除」（已掛載的分割區無法刪除）

### RAID Tab
- 顯示目前 RAID 狀態
- **建立 RAID**：精靈四步驟
  1. 選擇 RAID 等級（0/1/5/6/10）
  2. 選擇成員磁碟
  3. 選擇備用碟（可選）
  4. 確認建立

### fstab Tab
- 管理開機自動掛載設定
- 新增/刪除項目（系統掛載點 `/`、`/boot` 受保護）

### 使用歷史
- 頁面底部顯示磁碟使用量（progress bar）

## 檔案共享

### SMB 共享（Windows 網路芳鄰）

**新增共享：**
1. 檔案共享 → SMB 共享 → 「新增 SMB 共享」
2. 填入：
   - 名稱：共享顯示名稱（例如 `data`）
   - 路徑：NAS 上的資料夾路徑（例如 `/home/alex_chiang/share`）
   - 允許使用者：留空為全部，或填入帳號（逗號分隔）
   - 唯讀 / 允許訪客：依需求設定
3. 點「建立」

**Windows 連線：**
- 檔案總管輸入：`\\192.168.131.150\共享名稱`
- 輸入 NAS 的使用者帳號密碼

**編輯/刪除：** 共享建立後，表格每列有「編輯」「ACL」「刪除」按鈕

**ACL 權限：** 點「ACL」可設定每位使用者/群組的讀寫執行權限

### NFS 匯出（Linux 掛載）

**新增匯出：**
1. NFS 匯出 → 「新增 NFS 匯出」
2. 填入路徑和客戶端規則，例如：`192.168.1.0/24(rw,sync,no_subtree_check)`

**Linux 客戶端掛載：**
```bash
sudo mount 192.168.131.150:/path/to/export /mnt/nas
```

### 服務狀態
- 頁面頂部顯示 SMB/NFS 服務運行狀態和連線數

## 網路管理

### 網路介面
- 顯示所有介面的 IP、MAC、速率、狀態
- 點「設定」可切換 DHCP/Static（⚠️ 設錯可能斷線）

### 防火牆
- 查看/新增/刪除 iptables 規則
- SSH (port 22) 規則受保護，無法刪除

### 即時速率
- 每介面的上傳/下載速率
- 可開啟自動刷新（每 2 秒）

### 診斷工具
- Ping / Traceroute / DNS 查詢

### Wake-on-LAN
- 輸入目標 MAC 位址，發送喚醒封包

## 備份管理

### 備份任務
1. 點「建立備份任務」
2. 設定：名稱、來源路徑、目的地、排程（cron 格式）、保留天數
3. 可手動點「執行」立即備份
4. 點「歷史」查看執行紀錄
5. 從歷史中可「還原」到指定路徑

### Btrfs 快照
- 頁面底部「Btrfs 快照」區域
- 建立/查看/刪除快照（需 Btrfs 檔案系統）

## 常見問題

### Q: 格式化時找不到我的磁碟？
系統碟（掛載在 `/` 的磁碟）會被自動隱藏以防止誤操作。只有非系統碟且未掛載的裝置才會出現。

### Q: SMB 新增共享出現 Permission denied？
確認已設定 sudoers（見 README 的 Sudoers 設定章節），讓 backend 使用者可免密碼執行 Samba 相關指令。

### Q: 格式化出現 Permission denied？
1. 確認 sudoers 已設定 `mkfs` 免密碼執行
2. 確認裝置未被掛載

### Q: exFAT 格式化失敗 "No such file or directory"？
安裝 exfatprogs：`sudo apt install exfatprogs`

### Q: Windows 無法連線 SMB？
1. 確認 smbd 服務運行中（檔案共享頁面頂部狀態）
2. 確認使用者有 Samba 密碼：`sudo smbpasswd -a 使用者名稱`
3. 確認防火牆允許 port 445

### Q: 修改網路設定後斷線？
透過實體螢幕/鍵盤登入 NAS，或等待 netplan 自動回復，手動修正 `/etc/netplan/*.yaml`

## 系統需求

- **OS**: Debian 12 / Ubuntu 22.04+
- **RAM**: 512 MB (最低), 2 GB (建議)
- **Disk**: 200 MB (應用程式)
- **Python**: 3.11+
- **Node.js**: 18+

### 選用套件

```bash
sudo apt install smartmontools exfatprogs lm-sensors traceroute dnsutils nginx docker.io wireguard certbot btrfs-progs
```
